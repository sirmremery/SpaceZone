# 🚀 SpaceZone Gateway v10.0

یک دروازه (Gateway) سریع و مدرن برای تونل‌زنی VLESS روی WebSocket و XHTTP + HTTP Proxy، با داشبورد مدیریتی زیبا، ربات مدیریت تلگرام، صفحات ساب حرفه‌ای و قابلیت ساخت لینک‌های اختصاصی با محدودیت ترافیک، سرعت و آی‌پی.

---

## ✨ ویژگی‌ها

- 🔌 تونل VLESS روی چند ترابرد قابل‌انتخاب: **WebSocket**، **XHTTP (packet-up)** و **XHTTP (stream-up)**
- 🌐 HTTP Proxy داخلی
- 📊 داشبورد مدیریتی کامل (آمار، نمودار ترافیک ساعتی، اتصالات زنده، لاگ فعالیت‌ها و خطاها)
- 🔗 مدیریت لینک‌های نامحدود با محدودیت ترافیک اختصاصی (KB/MB/GB)
- 🚦 محدودیت سرعت (Bandwidth Throttling) اختصاصی برای هر کانفیگ، بر حسب Mbps
- ✅ فعال/غیرفعال‌سازی هر لینک به‌صورت لحظه‌ای، و انقضای خودکار بر اساس روز
- 📱 خروجی QR Code برای هر لینک و هر ساب
- 🛡️ Fingerprint (uTLS) و ALPN قابل تنظیم دستی برای هر کانفیگ
- 🔢 پورت اتصال قابل تنظیم دستی برای هر کانفیگ (نه فقط 443)
- 👥 محدودیت تعداد آی‌پی/کاربر هم‌زمان به‌ازای هر کانفیگ
- 🗂 **گروه‌های ساب**: چند کانفیگ رو داخل یک گروه بذار و یک لینک ساب حرفه‌ای و زیبا (صفحه‌ی عمومی، قابل رمزدار کردن) برای همه‌شون بگیر
- 💾 ذخیره‌سازی وضعیت روی دیسک (نه فقط حافظه) تا با ری‌استارت سرویس از بین نره — به شرطی که یک Volume دائمی روی مسیر دیتا وصل باشه
- 🤖 **ربات مدیریت تلگرام** (اختیاری) برای مدیریت کامل کانفیگ‌ها و گروه‌های ساب، بدون نیاز به باز کردن پنل وب
- 🎨 طراحی مدرن Glassmorphism با تم آبی-بنفش و دارک/لایت مود

## 🚀 نصب و راه‌اندازی

### 1️⃣ پیش‌نیازها

- Python 3.9 یا بالاتر
- pip
- حساب کاربری در [Railway.app](https://railway.app/) (یا هر هاست دیگری)

### 2️⃣ نصب وابستگی‌ها

```bash
pip install -r requirements.txt
```

### 3️⃣ اجرای محلی (برای تست)

```bash
python main.py
```

سرویس روی پورت `8000` اجرا می‌شود.  
برای تغییر پورت، متغیر محیطی `PORT` را تنظیم کنید.

## ☁️ Deploy روی Railway

1. ریپازیتوری رو به گیت‌هاب آپلود کن.
2. وارد **Railway.app** شو.
3. روی **New Project** → **Deploy from GitHub repo** کلیک کن.
4. ریپازیتوری رو انتخاب کن.
5. Railway به‌طور خودکار پروژه رو دیپلوی می‌کنه.
6. بعد از دیپلوی، یه **دامنه عمومی (Public Domain)** از تنظیمات Railway فعال کن.

برای اینکه اطلاعات با هر ری‌استارت از بین نره، یه **Volume** بساز و به مسیر `/data` متصل کن:

- تو تنظیمات سرویس → بخش **Volumes**
- یه Volume جدید با mount path `/data` ایجاد کن.

## 🔧 متغیرهای محیطی

| متغیر | توضیح | پیش‌فرض |
|---|---|---|
| `ADMIN_PASSWORD` | رمز ورود به پنل | `SpaceZone2025` |
| `SECRET_KEY` | کلید امنیتی (اختیاری) | خودکار ساخته می‌شه |
| `DATA_DIR` | مسیر ذخیره‌سازی | `/data` |
| `TELEGRAM_BOT_TOKEN` | توکن ربات تلگرام (اختیاری) | — |
| `TELEGRAM_ADMIN_IDS` | آیدی عددی ادمین‌ها با کاما (اختیاری) | — |
| `RAILWAY_PUBLIC_DOMAIN` | دامنه عمومی | `localhost` |
| `RATE_LIMIT_PER_MINUTE` | محدودیت درخواست در دقیقه | `60` |
| `PORT` | پورت اجرا | `8000` |

## 📱 نحوه استفاده از پنل

### 🖥️ ورود به پنل

1. برو به آدرس `https://your-domain.railway.app/dashboard`
2. رمز پیش‌فرض: `SpaceZone2025`
3. بعد از ورود، داشبورد رو می‌بینی.

### 🔗 ساخت کانفیگ جدید

1. تو داشبورد، برو به بخش **کانفیگ‌ها**.
2. روی **ساخت کانفیگ جدید** کلیک کن.
3. اطلاعات رو وارد کن:
   - **برچسب**: یه اسم براش انتخاب کن
   - **پروتکل**: `VLESS/WS` یا `XHTTP`
   - **Fingerprint**: `chrome`، `firefox` و ...
   - **ALPN**: `http/1.1`، `h2,http/1.1` و ...
   - **پورت**: عددی بین 1 تا 65535
   - **محدودیت حجم**: بر حسب KB/MB/GB (0 = نامحدود)
   - **محدودیت سرعت**: بر حسب Mbps (0 = نامحدود)
   - **محدودیت آی‌پی**: تعداد کاربر هم‌زمان (0 = نامحدود)
   - **انقضا**: تعداد روز (0 = بدون انقضا)
4. روی **ساخت کانفیگ** کلیک کن.

### 📋 دریافت لینک اتصال

- از کارت کانفیگ، روی دکمه **کپی لینک** کلیک کن.
- یا از دکمه **QR** برای دریافت کد QR استفاده کن.

### 🗂 مدیریت گروه‌های ساب

1. برو به بخش **گروه‌های ساب**.
2. روی **گروه جدید** کلیک کن.
3. نام و توضیحات گروه رو وارد کن (اختیاری: رمز صفحه).
4. بعد از ساخت، می‌تونی کانفیگ‌ها رو به گروه اضافه کنی.
5. لینک ساب حرفه‌ای گروه رو کپی کن و به اشتراک بذار.

## 🤖 ربات تلگرام

### راه‌اندازی

1. از [@BotFather](https://t.me/BotFather) یه توکن بگیر.
2. متغیرهای محیطی رو تنظیم کن:
   - `TELEGRAM_BOT_TOKEN` = توکن ربات
   - `TELEGRAM_ADMIN_IDS` = آیدی عددی تلگرام خودت (مثلاً `123456789`)
3. ری‌دپلوی کن.
4. به ربات پیام `/start` بده.

### دستورات ربات

| دستور | توضیح |
|-------|--------|
| `/start` | منوی اصلی |
| `/menu` | منوی اصلی |
| `/help` | راهنمای کامل |
| `/cancel` | لغو عملیات جاری |
| `/stop` | لغو عملیات جاری |
| `/stats` | نمایش داشبورد آماری |
| `/report` | گزارش کامل از تمام کانفیگ‌ها |

### امکانات ربات

- 📋 لیست، مشاهده، فعال/غیرفعال‌سازی و حذف کانفیگ‌ها
- ➕ ساخت کانفیگ جدید با ویزارد مرحله‌به‌مرحله
- 🔗 دریافت لینک اتصال VLESS و لینک ساب
- 🗂 ساخت و مدیریت گروه‌های ساب
- 📊 دریافت آمار لحظه‌ای
- 🔑 تغییر رمز پنل از طریق ربات

## 🔒 امنیت

- **احراز هویت**: کوکی `HttpOnly` + `Secure` + `SameSite=Lax`
- **رمز عبور**: هش SHA-256 با Salt
- **محدودیت نرخ درخواست**: جلوگیری از حملات Brute Force
- **CORS**: محدود به دامنه‌های مجاز
- **هدرهای امنیتی**: `X-Frame-Options: DENY`، `X-Content-Type-Options: nosniff`
- **Session TTL**: 7 روز (قابل تنظیم)
- **UUID Authentication**: سخت‌گیرانه

## 📊 API Endpoints

| متد | مسیر | توضیح |
|-----|------|--------|
| `POST` | `/api/login` | ورود به پنل |
| `POST` | `/api/logout` | خروج از پنل |

| `GET` | `/api/connections` | اتصالات فعال |
| `GET` | `/api/activity` | لاگ فعالیت‌ها |

## 🐛 عیب‌یابی

### خطای `409 Conflict` در ربات تلگرام

این خطا یعنی یه نمونه دیگه از ربات در حال اجراست. برای رفع:

```bash
# تو Railway، سرویس رو ری‌استارت کن
# یا مطمئن شو فقط یه نمونه در حال اجراست
```

### خطای `404 Not Found` برای لوگو

لینک لوگو رو تو `pages.py` به‌روز کن:

```python
LOGO_URL = "https://your-image-url.com/logo.jpg"
```

### خطای `401 Unauthorized`

- مطمئن شو رمز عبور درسته.
- کوکی مرورگر رو پاک کن.
- دوباره وارد شو.
